02、安装常用插件

  • Folders Plugin : 该插件允许用户创建“文件夹”来组织任务，下载地址为: https://plugins.jenkins.io/cloudbees-folder/, 需要下载和jenkins对应的版本安装，否则会安装失败。
  • git: 安装git之后可以从git仓库将代码clone到本地，这个在开发中比较重要
