Java学习笔记-I/O

Java中传统的java.io包基于流模型实现，包括字节流和字符流，字节核心类包括输入流java.io.InputStream和输出流java.io.OutputStream。字符流核心类包括java.io.Reader和java.io.Writer。

一、java I/O概述

  • 流(Stream)是指数据在计算机与计算机之间或计算机各部件之间的流动，例如: 数据从硬盘流入到内存、数据通过网络在不通计算机之间流动等。
